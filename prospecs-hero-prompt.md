> 재사용 프롬프트. 아래 **①템플릿**을 복사해 브랜드 변수만 갈아끼우면 된다.

## ② 나이키 데모 대비 보완한 7가지 (설계 결정)

| # | 나이키 데모의 문제 | 프로스펙스 판 해결 |
|---|---|---|
| ① | 카피·CTA가 스크롤 86%에서야 등장 → 첫 화면에 진입점 0 | `P=0`에서도 상단 브랜드 라인 + 하단 **'컬렉션 바로가기'** 앵커 상시 노출. 스크롤은 이걸 대형 카피로 *승격*만 함 |
| ② | 실텍스트가 `sr-only` h1 하나뿐 → 크롤러가 볼 게 없음(GEO 취약) | h1 실제 렌더 + 히어로↔컬렉션 사이 **브랜드 소개 문단**을 실텍스트로 |
| ③ | display 폰트가 `Futura` → 한국 기기엔 없어 `Impact`로 폴백, 렌더 붕괴 | display를 **한글 안전 스택**(Pretendard→Apple SD Gothic Neo→Malgun Gothic)으로. 헤드라인을 한글로 |
| ④ | `prefers-reduced-motion`에서 카메라만 멈추고 canvas는 계속 렌더 | reduced면 **완전 정적**: 조형 1구도 고정(P=0.52) + 파티클 0 + 카피 전부 즉시 표시 + 스크림 상시 |
| ⑤ | 카피가 canvas 위에 뜨는데 대비 보장 장치 없음 | 카피 뒤 **radial 스크림** 상시(조형이 밝아도 흰 텍스트 확보) |
| ⑥ | 나이키도 레드, 프로스펙스도 레드 → 그대로면 두 브랜드가 똑같이 보임 | **컨셉 축 자체를 분리** (아래 ③) |
| ⑦ | 디렉터 패널 + 다크→라이트 seam 접합은 훌륭 | **그대로 유지** — 실제 사이트(sky/navy 라이트)와의 접합을 보여주는 장치 |

## ③ 프로스펙스 컨셉 — "열간 단조(Forge)"

나이키 = *차가운 크롬 · 우주적 암전 · 곡선 안으로 파고드는 카메라*
프로스펙스 = **강철을 붉게 벼려내는 열간 단조 · 내부 발열(emissive) · 위로 비상하는 이중 날개**

- **왜 이 축인가**: 프로스펙스 정체성(토종·장인정신·"우리의 속도는 계속"·"직진 그날까지")과
  대장간에서 강철을 벼려 날을 세우는 은유가 맞물린다. 40여 년 헤리티지 = 벼려온 시간.
- **라이팅**: 크롬 반사(나이키) 대신 조형 코어에서 **붉게 달아오르는 발열**. 스크롤이 진행될수록
  강철이 뜨거워진다(`heat = seg(P,.12,.82)`). 채움이 아니라 코어 발광 + 엣지 글로우.
- **카메라**: 나이키는 곡선 안으로 파고들었다(panX 음수). 프로스펙스는 **위로 솟는다**(panY 음수) = 비상·상승.
- **파티클**: 먼지(나이키) → **대장간 불티**가 아래에서 위로 떠오름(달아오를수록 밝아짐).
- **배경**: 순흑(#050505) → **딥 네이비**(#0a0f1a)로 분리.
